
## How to use the script

```
TELEGRAM_CHAT_ID=
TELEGRAM_TOPIC_ID=
TELEGRAM_MESSAGE=
TELEGRAM_BOT_TOKEN=
curl -L -s https://raw.githubusercontent.com/duonghd7/telegram_notification/master/telegram_notification.sh | bash /dev/stdin "$TELEGRAM_CHAT_ID" "$TELEGRAM_TOPIC_ID" "$TELEGRAM_MESSAGE" "$TELEGRAM_BOT_TOKEN"
```
